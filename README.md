Title: Automated EC2 Instance Management Using AWS Lambda

Objective:
To automate the starting and stopping of EC2 instances based on tags.

Steps Followed:
1. Created two EC2 instances.
2. Tagged instances as Auto-Start and Auto-Stop.
3. Created IAM role with AmazonEC2FullAccess.
4. Developed Lambda function using Python and Boto3.
5. Implemented logic to identify instances based on tags.
6. Started and stopped instances accordingly.
7. Tested and verified results.

Result:
EC2 instances were successfully started and stopped based on tags.

Conclusion:
Automation improves operational efficiency and reduces manual effort.
